Tentang SiOSS

Informasi sistem dan sumber data.

Apa itu SiOSS?

SiOSS adalah Sistem Inventarisasi Online Zona Selamat Sekolah yang digunakan untuk menampilkan, mengelola, dan memvisualisasikan data sekolah beserta status implementasi Zona Selamat Sekolah secara online.

Website ini dibangun menggunakan teknologi frontend murni agar mudah di-host di GitHub Pages tanpa memerlukan server backend.

Teknologi yang Digunakan

  • HTML5, CSS3, JavaScript
  • Bootstrap 5
  • DataTables
  • Chart.js
  • Leaflet.js
  • Google Spreadsheet CSV sebagai sumber data
